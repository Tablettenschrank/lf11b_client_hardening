# Secure Boot

Secure Boot zu aktivieren sorgt dafür, dass während des Bootvorganges kein Schadcode ausgeführt werden kann. 

(Die Umsetzung von Secure Boot fehlt aus zeitlichen Gründen, wäre aber umgesetzt worden, wenn es mehr Zeit gegeben hätte.)

Um Secure Boot nutzen zu können, ist es nötig, UEFI zu verwenden. Laut Internet scheint das BIOS meist keine Secure-Boot-Unterstützung zu haben. UEFI ist im Vergleich zum BIOS eine Art Weiterentwicklung mit mehr Funktionen. BIOS ist eine Firmware, während UEFI ein eigenes Betriebssystem ist.
